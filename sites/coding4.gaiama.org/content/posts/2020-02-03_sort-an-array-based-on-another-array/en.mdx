# ES6: Sort an array based on another array

## Array of Numbers

This is more to show the basic idea

```js size=content
const array = [1, 2, 3, 4, 5]
const sortArray = [3, 5, 2, 4, 1]

const arrayMap = array.reduce(
  (accumulator, currentValue) => ({
    ...accumulator,
    [currentValue]: currentValue,
  }),
  {}
)

const result = sortArray.map(key => arrayMap[key])
// [3, 5, 2, 4, 1]
```

## Array of Objects

That's the case I needed it for and am using it to sort the `TODO` comments on the [Roadmap page](/en/roadmap).

It's almost the same, note how only the key `[currentValue.key]` differs and has to be present in the array objects.

```js size=content
const array = [
  { key: 'foo', value: 'bar' },
  { key: 'foobar', value: 'baz' },
  { key: 'qux', value: 'quux' },
]
const sortArray = ['foobar', 'qux', 'foo']

const arrayMap = array.reduce(
  (accumulator, currentValue) => ({
    ...accumulator,
    [currentValue.key]: currentValue,
  }),
  {}
)

const result = sortArray.map(key => arrayMap[key])
// [
//   { key: 'foobar', value: 'baz' },
//   { key: 'qux', value: 'quux' },
//   { key: 'foo', value: 'bar'}
// ]
```
